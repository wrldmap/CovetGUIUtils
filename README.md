# CovetGUIUtils
### A really bad and overcomplicated plugin for something that could have been done way easier.
### Supports Paper on 1.21.10 (Other versions untested, experiences may vary.)

This plugin provides utilities for making custom GUIs for Minecraft servers using a HOCON-based configuration system. It supports ItemsAdder, Oraxen, and (hopefully) Nexo.
